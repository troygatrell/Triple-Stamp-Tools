Apparel Production Checklist
  • Find sample
  • Tape screens
  • Register job (rough)
  • Move pallets
  • Set off-contact
  • Adjust squeegee depth, pressure, and angle
  • Clean/glue pallets
  • Print registration screen and line up
  • Set dryer temperature and speed
  • Adjust flashes (Is this a high-temp flash job?)
  • Adjust stamp temperature
  • Program press
  • Check Trello for any additional notes
  • Mix ink
  • Find scrap shirts
  • Test print
  • Double-check proof against test print
  • Measure artwork size
  • Take notes on setup
  • Check proof and place it at the back of the dryer
  • Verify dryer speed
  • Determine who is catching
  • Check ShopVox flags
  • Move card in Trello
  • Return or save sample
